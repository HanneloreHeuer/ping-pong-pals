Ping Pong Pals

Welcome! This website is dedicated to keeping track of the outcomes of friendly ping pong matches using the Elo rating system, and to encourage everyone to join the game in a fun and competitive way.

This website is still in an early development phase, there will be bugs. The code is open source, licensed under the MIT license. If you would like to contribute, visit the Github page.

Elo

We use the Elo rating system to keep track of a player's skill relative to other players. Your score is updated after every game you play, based on how much the game outcome deviates from the expected outcome. To learn more about the Elo rating system, visit the Wikipedia page.
